# Best-Hotel

Best-Hotel is a modern hotel website built with Django. The project includes a responsive interface, an online booking page, and an admin panel for managing reservations.

Repository: https://github.com/Am0nik/Best-Hotel

## Features

- Responsive design
- Modern hotel landing page
- About section
- Photo gallery
- Customer reviews
- Booking page
- Booking success page
- Admin login
- Reservation management

## Run

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
